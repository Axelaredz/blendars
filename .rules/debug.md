# Debug Protocol

> `/debug` или `/debug full` в конце сообщения → добавь блок после ответа.
> Без команды — игнорируй этот файл полностью.

## Главное правило: ЦИТИРУЙ ИЛИ МОЛЧИ

- Заявление «применил правило» ОБЯЗАНО содержать точную цитату из .rules/
- Не можешь найти цитату → пиши [встроенное поведение]
- Пользователь проверит grep-ом. Выдумка = бесполезный debug
- НЕ выдумывай правила, которых нет
- Действовал по дефолту → так и пиши, не приписывай файлу

## Протокол

1. Удали `/debug` (`/debug full`) из текста
2. Ответь на запрос как обычно
3. ПОСЛЕ ответа добавь блок ниже
