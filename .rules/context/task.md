# Task: текущая задача

## Goal
Создать тестовую сцену для UI-kit: набор базовых UI-компонентов
в киберпанк-аниме стилистике, размещённых на одном экране
для визуальной проверки и итерации.

## Scope
- Сцена: `res/ui/test_ui_kit.tscn`
- Компоненты для тестирования:
  - Button (primary, secondary, danger)
  - Label (heading, body, caption)
  - Panel (container с неоновым border)
  - HealthBar / ProgressBar
  - Input field
  - Tooltip
- Каждый компонент — отдельная переиспользуемая сцена в `res/ui/components/`
- Тестовая сцена собирает все компоненты на одном экране с подписями

## Acceptance Criteria
- Все компоненты видны на одном экране без скролла (или с минимальным)
- Стиль соответствует `core/styleguide.md`
- Каждый компонент — отдельный .tscn, подключается через instance
- Кнопки реагируют на hover/press (визуальный feedback)
- Тёмный фон, неоновые акценты

## Out of Scope
- Функциональная логика UI (инвентарь, чат)
- Подключение к серверу
- Финальные ассеты — заглушки допустимы
